# Coders for Coders (CFC) Community Hub

Welcome to the official repository for the **Coders for Coders** community website. Built with Next.js and styled with the premium **CFC Obsidian** aesthetic.

🚀 **Live Production:** [https://cfc-ashen.vercel.app](https://cfc-ashen.vercel.app)

## ✨ Features

- **About Us**: Discover our mission and the story behind CFC.
- **Projects Showcase**: Explore open-source projects built by the community.
- **Community Journal**: Deep dives into tech, design, and member stories.
- **Community Events**: Stay updated with hackathons, workshops, and meetups.
- **Join Flow**: An interactive onboarding experience for new members.

## 🛠 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/coders-for-coders/site-new.git
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Run the development server:
   ```bash
   pnpm run dev
   ```
